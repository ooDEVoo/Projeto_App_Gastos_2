# Controle de Gastos

Aplicação desktop desenvolvida em Python para controle de gastos pessoais, com interface gráfica, banco de dados SQLite, filtros e gráficos integrados.

## Funcionalidades

- Cadastro de lançamentos financeiros
- Edição de lançamentos
- Exclusão de lançamentos
- Filtro por mês e ano
- Filtro por tipo
- Filtro por categoria
- Busca por texto
- Resumo financeiro
- Cálculo do saldo do mês atual
- Gráfico de despesas por categoria
- Visualização em gráfico de barras ou pizza

## Tecnologias utilizadas

- Python
- Tkinter
- SQLite
- Matplotlib
